#Old Mountain

##About Old Mountain

Old Mountain is a novel by Shay Solette, being written for[http://nanowrimo.org](National Novel Writing
Month) in 2014. It is set in a [https://en.wikipedia.org/wiki/Low_fantasy](low-fantasy) world, and is draws
inspiration from old western movies.
